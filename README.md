### Installation

1. Install Beautiful Soup
```
  pip install beautifulsoup4
```
2. Install Pandas
```
 pip install pandas
```
3. Install Splinter
```
 pip install splinter
```
4. Install Jupyter
```
$ pip install jupyter
```
5. Run Jupyter
```
 Jupyter Notebook
```
